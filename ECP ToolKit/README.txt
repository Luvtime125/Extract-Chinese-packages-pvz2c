ECP (Extract Chinese packages)
Version 2.0
coded with python script and json files

What is new?

1. its work with any path just like ECP.py in disk C and ipa in disk D
2. show rsb names and extracted files number
3. you can select language like of (English,spanish,chinese) by selected_language.json 
by replacing

{
  "selected_language": "en"
}

English "en"
Spanish "es"
Chinese "zh" 

Note : you must have python 3.6 or higher 
and you can use library_setup for install library
